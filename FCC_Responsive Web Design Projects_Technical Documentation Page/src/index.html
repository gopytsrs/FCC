Introduction

Python is a programming language that lets you work quickly and integrate systems more effectively.

Python can be easy to pick up whether you're a first time programmer or you're experienced with other languages. The following pages are a useful first step to get on your way writing programs with Python!

Variables

Python has no command for declaring a variable. A variable is created the moment you first assign a value to it.

Example: x = 5, y = "John"

Variable naming conventions

  • A variable name must start with a letter or the underscore character
  • A variable name cannot start with a number
  • A variable name can only contain alpha-numeric characters and underscores (A-z, 0-9, and _ )
  • Variable names are case-sensitive (age, Age and AGE are three different variables)

Data types

Python has the following data types built-in by default, in these categories:

  • Text Type: str
  • Numeric Types: int, float, complex
  • Sequence Types: list, tuple, range
  • Mapping Type: dict
  • Set Types: set, frozenset
  • Boolean Type: bool
  • Binary Types: bytes, bytearray, memoryview

Operators

There are many different types of operators in Python, some examples are Arithemtic, Comparison and Logical operators

Arithmetic operators:

Used for arithmetic operations.

Operator Name Usage
+ Addition 2 + 1
- Subtraction 2 - 1
* Multiplication 2 * 1
/ Division 2 / 1

Comparison Operators

Used for comparison between variables or data types.

Operator Name Usage
> Greater than x > y
< Lesser than x < y
== Equal to x == y
!= Not equal to x != y

Logical operators

Used to combine conditional statements

Operator Description Usage
and Returns true if both sides are true x and y
or Returns true if one side is true x or y
not Reverse the boolean value not x

Conditionals

Conditionals are conditional statements, to choose between different options.There are three type of conditionals, namely if, elif and else.

if

Checks whether the condition is true, if it is, it executes the code after the block.

Example:

if 2 > 3:
  print("Hello")

else

Typically used in conjuction with if, the code in the else block executes if the if statement returns false.

Example:

if 2 > 3:
  print("Hello")
else:
  print("Else")

elif

Typically used if there is one or more condiiton to check.

Example:

if 2 > 3:
  print("Hello")
elif 2 == 3:
  print("Impossible")
